# Portfolio — มนัสนันท์ มาตรา

เว็บไซต์ Portfolio ธีมสีแดง-ดำแนวปีศาจ ทำด้วย HTML/CSS/JavaScript ล้วน (ไม่ต้องใช้ Framework)
พร้อมอัปโหลดขึ้น GitHub Pages ได้ทันที

## โครงสร้างไฟล์

```
├── index.html        หน้าหลัก
├── education.html    ประวัติการศึกษา
├── personal.html      ประวัติส่วนตัว
├── award.html         ผลงาน (แกลเลอรีคลิกดูรูปขยาย)
├── activities.html    กิจกรรม (แกลเลอรีคลิกดูรูปขยาย)
├── contact.html       ช่องทางติดต่อ
├── css/style.css      สไตล์ทั้งหมดของเว็บไซต์
├── js/script.js        เมนูมือถือ + เอฟเฟกต์ประกายไฟ + lightbox
└── images/            รูปภาพที่ใช้ในเว็บไซต์
```

## สิ่งที่ต้องแก้ไขก่อนส่งงาน

ทุกจุดที่เขียนว่า **"แก้ไขตรงนี้"** ในไฟล์ HTML คือจุดที่ต้องใส่ข้อมูลจริงของคุณ:

1. **personal.html** — ชื่อเล่น วันเกิด โรงเรียน คติประจำใจ ความสามารถพิเศษ
2. **education.html** — ปีการศึกษา ชื่อโรงเรียนแต่ละช่วงชั้น เกรดเฉลี่ย
3. **award.html / activities.html** — ชื่อผลงาน/กิจกรรม ปี หน่วยงาน และเปลี่ยนรูปภาพ
4. **contact.html** — อีเมล เบอร์โทร Facebook Instagram Line GitHub

## วิธีเปลี่ยนรูปภาพ

1. นำไฟล์รูปใหม่ไปวางในโฟลเดอร์ `images/`
2. แก้ไข `src="images/ชื่อไฟล์.jpg"` ในแท็ก `<img>` ให้ตรงกับชื่อไฟล์ใหม่
3. สำหรับหน้า award.html และ activities.html แต่ละการ์ดจะมี 2 จุดที่ต้องแก้ให้ตรงกัน:
   - `<img src="images/...">` (รูปตัวอย่างเล็ก)
   - `data-full="images/...` (รูปที่จะแสดงตอนคลิกขยาย — ควรใช้ไฟล์เดียวกันหรือไฟล์ความละเอียดสูงกว่า)

รูปภาพขนาดใหญ่เกินไปจะทำให้เว็บโหลดช้า แนะนำให้บีบอัดไฟล์ให้มีขนาดไม่เกิน 500 KB ต่อรูป
ก่อนอัปโหลด (ใช้เว็บไซต์บีบอัดรูปฟรี เช่น squoosh.app หรือ tinypng.com)

## วิธีอัปโหลดขึ้น GitHub Pages

1. สร้าง repository ใหม่บน GitHub (หรือใช้ repo เดิม `portfolio`)
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้นไปที่ repo (ต้องมี `index.html` อยู่ที่ root ของ repo)
3. ไปที่ Settings → Pages → เลือก Branch เป็น `main` และ Folder เป็น `/ (root)` แล้วกด Save
4. รอสักครู่ เว็บไซต์จะขึ้นออนไลน์ที่ `https://ชื่อผู้ใช้.github.io/ชื่อ-repo/`

## หมายเหตุ

- แบบฟอร์มติดต่อในหน้า contact.html เป็นเพียงตัวอย่าง UI เท่านั้น
  เนื่องจาก GitHub Pages เป็นการโฮสต์แบบ Static ไม่มีระบบส่งอีเมลจริง
  หากต้องการให้ใช้งานได้จริง ต้องเชื่อมกับบริการภายนอก เช่น Formspree หรือ Google Form
